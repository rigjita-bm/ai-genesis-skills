# 📧 Email Responder

Автоматические ответы на email естественным языком через Gmail API.

## Что делает

- Читает непрочитанные письма из Gmail
- Определяет тип письма (запрос/жалоба/партнёрство/поддержка/спам)
- Генерирует персонализированный ответ в выбранном тоне
- Сохраняет черновики или отправляет автоматически

## Установка

```bash
# Убедитесь что gog skill настроен (Gmail API)
# Проверьте credentials в /root/.openclaw/skills/skills/gog/
```

## Использование

### В коде:
```python
from email_responder import EmailResponder

# Создаём респондер с дружелюбным тоном
responder = EmailResponder(tone="friendly")

# Проверяем почту и создаём ответы
responses = responder.check_and_reply(auto_send=False)

# Отправляем вручную после проверки
for item in responses:
    print(item['response']['body'])
```

### Типы писем:

| Тип | Ключевые слова | Стратегия ответа |
|-----|----------------|------------------|
| inquiry | интересует, цена, как заказать | Ответить на вопросы, предложить созвон |
| complaint | жалоба, проблема, возврат | Извиниться, предложить решение |
| partnership | сотрудничество, партнёрство | Выразить интерес, запросить детали |
| support | помощь, не работает, вопрос | Дать инструкцию, предложить поддержку |
| spam | рассылка, реклама, выигрыш | Игнорировать |

### Тоны ответа:

- `professional` — Деловой (формальный)
- `friendly` — Дружелюбный (по умолчанию)
- `casual` — Неформальный (для знакомых)

## Примеры

### Входящее письмо:
```
От: Иван <ivan@company.com>
Тема: Стоимость автоматизации

Добрый день! У меня салон красоты. 
Сколько стоит настроить бота для записи клиентов?
```

### Сгенерированный ответ:
```
Привет, Иван!

Спасибо за интерес к нашим услугам!

По вашим вопросам:
• Сколько стоит настроить бота для записи клиентов? — отвечу подробно в ближайшее время.

Давайте созвонимся на 10-15 минут? Это быстрее, чем переписка. Подскажите, когда вам удобно?

До связи,
AI Genesis Team
```

## Безопасность

⚠️ **Никогда не включайте `auto_send=True` без проверки!**

Рекомендуемый workflow:
1. Сначала `auto_send=False` — получить черновики
2. Проверить ответы
3. Отправить вручную или настроить авто для доверенных отправителей
