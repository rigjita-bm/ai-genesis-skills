# 🔄 Hybrid Approach: Notion + Local Files

**AI Genesis — Гибридная архитектура**

---

## Концепция

```
┌─────────────────────────────────────────────────────────────┐
│  NOTION — ИСТОЧНИК ПРАВДЫ (редактируете здесь)              │
│  ├── Цены и тарифы                                          │
│  ├── Шаблоны сообщений (A/B/C)                             │
│  └── Конфигурации                                           │
└─────────────────────────────────────────────────────────────┘
                              ↓ genesis sync
┌─────────────────────────────────────────────────────────────┐
│  ЛОКАЛЬНЫЕ ФАЙЛЫ — ИСПОЛНЕНИЕ (быстро, надёжно)            │
│  ├── Кэш цен (~/.openclaw/cache/prices.json)               │
│  ├── Кэш шаблонов (~/.openclaw/cache/templates.json)       │
│  └── Обновлённые скрипты                                   │
└─────────────────────────────────────────────────────────────┘
```

---

## Быстрый старт

### 1. Синхронизация (одной командой)

```bash
# Полная синхронизация
genesis sync all

# Только цены
genesis sync config

# Только шаблоны
genesis sync templates

# Проверить статус
genesis sync status
```

### 2. Что синхронизируется

| Данные | Источник | Куда | Частота |
|--------|----------|------|---------|
| Цены | Notion DB | proposal-generator | При изменении |
| Шаблоны follow-up | Notion DB | followup-system | При изменении |
| Конфигурации | Notion DB | Все навыки | При изменении |

---

## Структура в Notion

### База 1: "Config — Цены"

Создайте database с полями:
- `Name` (Title): audit_price, pilot_price, full_price, combo_price
- `Value` (Number): 100, 350, 700, 1000
- `Last Updated` (Date): авто

### База 2: "Templates — Follow-up"

- `Day` (Select): Day 1, Day 3, Day 5, Day 7
- `Variant` (Select): A, B, C
- `Channel` (Select): text, voice, ig
- `Content` (Text): Текст шаблона
- `Active` (Checkbox): ✅

## Преимущества подхода

### ✅ Скорость
- Локальные файлы — мгновенный доступ
- Нет задержек API при работе навыков
- Кэш работает даже без интернета

### ✅ Надёжность
- Notion падает — навыки работают (из кэша)
- Сервер перезагружается — ничего не теряется
- Автоматическое резервирование

### ✅ Контроль
- Меняете в Notion → везде обновляется
- История изменений в Notion
- Версионирование через sync

---

## Рабочий процесс

### Когда меняете цены:

1. **В Notion**: Меняете pilot_price с 350 на 400
2. **В терминале**: `genesis sync config`
3. **Готово**: Все КП теперь с новой ценой

### Когда тестируете шаблоны:

1. **В Notion**: Редактируете Day 1, Variant A
2. **В терминале**: `genesis sync templates`
3. **Тест**: `genesis followup send "Test" --variant A`

## Технические детали

### Кэш
```
~/.openclaw/cache/
├── prices.json        # Последние цены
└── templates.json     # Шаблоны сообщений
```

### Конфиги навыков
```
~/.openclaw/skills/skills/proposal-generator/
├── proposal_generator.py
└── notion_config.py   # ← Авто-генерация
```

### Cron (авто-синхронизация)
```bash
# Синхронизировать каждые 6 часов
0 */6 * * * cd /root/.openclaw && bash genesis sync all > /dev/null 2>&1
```
