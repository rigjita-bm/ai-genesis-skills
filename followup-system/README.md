## 🎯 Команды

### `genesis followup list [опции]`
Показывает всех лидов, отсортированных по приоритету.

**Опции:**
- `--today` — только срочные на сегодня
- `--week` — на неделю
- `--urgent` — только 🔴

**Вывод:**
```
Приоритет  | Имя    | Ниша    | Статус | Дней | Скоринг | Действие
🔴 КРИТИЧНО | Анна   | salon   | 🆕 new | 2    | 85      | Немедленное касание
```

### `genesis followup hot [опции]`
Показывает горячих лидов (скоринг ≥70, молчат ≥3 дней).

**Опции:**
- `--min-score 80` — кастомный порог скоринга
- `--min-silent 5` — кастомный порог молчания

### `genesis followup stats`
Дашборд с метриками:
- 🔥 Лидов требуют внимания
- ⏳ В работе
- ❄️ Холодных
- 💰 Pipeline value
- 📈 Конверсия
- ⏱️ Среднее время ответа
- 🎯 Прогноз на месяц

### `genesis followup next "Имя"`
Показывает следующий шаг для конкретного лида.

**Пример:**
```bash
genesis followup next "Иван"
```

### `genesis followup send "Имя" --variant A`
Отправляет сообщение (A/B/C) и логирует.

**Примеры:**
```bash
genesis followup send "Иван" --variant A
genesis followup send "Мария" --variant B
```

### `genesis followup archive "Имя"`
Архивирует лида в ⚪ **АРХИВ** — финальный статус, не возвращаем.

**Пример:**
```bash
genesis followup archive "Иван"
```

## ✉️ Шаблоны сообщений

Для каждого касания генерируется 3 варианта:

**День 1 (знакомство):**
- A: Прямой вопрос о боли
- B: Интрига через кейс
- C: Конкретный вопрос по времени

**День 3 (кейс):**
- A: Социальное доказательство
- B: Обучение (3 ошибки)
- C: Сравнение с конкурентами

**День 5 (возражения):**
- A: Прямой вопрос "что останавливает"
- B: Мягкое развеивание
- C: Альтернатива (пилот вместо полного)
